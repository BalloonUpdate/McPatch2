### McPatch2

McPatch第二版的管理端的源代码（也包括web页面源代码）。

### crates说明

| 名称                   | 用途                                                         |
| ---------------------- | ------------------------------------------------------------ |
| manager        | 管理端主程序。负责更新包的打包和管理工作，也提供内置开箱即用的内置服务端 |
| xtask                  | 用于ci/cd自动化打包的行为和命令                              |

### 常用命令说明

| 命令                                | 作用                                 |
| ----------------------------------- | ------------------------------------ |
| `cargo dev`                          | 开发场景下，启动管理端程序进行测试           |
| `cargo ci`                            | 自动构建场景下，打包管理端 |
